Publications

  • POC-CSP: Parameterised and Orthogonally-Constrained Neural Network

    Neural Networks Journal • 2024

    Authors: Milad Soleymani, Mohammad Ali Ahmadi, Mehdi Ezoji

    Abstract: We propose POC-CSP, a novel neural network layer that learns Common Spatial Patterns (CSP) with enhanced orthogonality constraints. This approach improves the discriminative power of spatial filters for EEG-based brain-computer interfaces (BCIs) by incorporating parameterized constraints directly into the neural architecture.

    Key Contributions:

    • • Novel parameterized orthogonality constraints for CSP learning
    • • Integration with deep neural architectures
    • • State-of-the-art performance on multiple BCI datasets
    • • Maintains interpretability of learned spatial patterns

  • A Self-supervised Task-agnostic Embedding for Multi-task EEG Classification

    Under Review • 2024

    Authors: Milad Soleymani, Hamed Azami, Ali Motie Nasrabadi

    Abstract: This paper presents a self-supervised learning framework for creating universal EEG representations that generalize across different BCI tasks. By leveraging contrastive learning and temporal consistency, our approach learns task-agnostic features from raw EEG signals without requiring labeled data.

    Key Contributions:

    • • Self-supervised learning framework for EEG signals
    • • Task-agnostic representations for multiple BCI applications
    • • Significant performance improvements on downstream tasks
    • • Transfer learning across motor imagery, emotion recognition, and cognitive load

  • A Deep Learning Algorithm for Classifying Grasp Motions using Multi-session EEG Recordings

    IEEE Access • 2021

    Authors: Mohammad Ali Ahmadi, Milad Soleymani, Mehdi Ezoji

    Abstract: We present a novel deep learning architecture for classifying grasp motions from EEG signals recorded across multiple sessions. Our approach addresses the challenge of inter-session variability through domain adaptation techniques and temporal convolutional networks.

    Key Contributions:

    • • Novel deep learning architecture for multi-session EEG
    • • Domain adaptation for inter-session variability
    • • 87.3% accuracy on multi-session dataset
    • • Applications in long-term BCI systems for motor rehabilitation
